# Life Assistant — Weekly Planner

A lightweight weekly planner for managing family meals, shopping, and day-to-day logistics. Built as a single HTML file — no installation, no account, no backend.

**[Open the app](https://jdgyp.github.io/life-assistant)**

## What it does

- Plan meals (breakfast, lunch, dinner) for each day of the week
- Add notes and tasks per day
- Manage a shopping list with progress tracking
- Navigate between weeks
- All data saves automatically in your browser

## Why I built this

As a Product Manager moving into AI-assisted product development, I built this to demonstrate that modern AI tools allow PMs to ship working software — not just write about it. This app was built with [Claude Code](https://claude.ai/code).

## Run it locally

Download `index.html` and open it in any browser. No build step, no dependencies.
